🔧 Serveur de Développement :

Pour utiliser un serveur local avec rechargement automatique :

# Installer live-server (si pas déjà fait)
npm install -g live-server

# Démarrer le serveur
live-server --port=3000

Puis visitez : http://localhost:3000
